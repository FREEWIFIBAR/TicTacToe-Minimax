# TicTacToe-Minimax

Programa en Haskell que implementa el clásico juego del Tres en Raya con dos modos de juego:

- Jugador contra Jugador
- Jugador contra Máquina

El agente inteligente está implementado mediante el algoritmo Minimax, el cual analiza
exhaustivamente el espacio de estados del juego y selecciona siempre las jugadas óptimas,
asumiendo un oponente perfecto. Gracias a esto, la IA garantiza que nunca perderá
(únicamente puede ganar o empatar).
